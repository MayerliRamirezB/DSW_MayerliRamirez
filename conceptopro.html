    • Propiedades bàsicas de CSS


    1. Font-family: Define la familia tipográfica.Se tiene 3 opciones para definir la elección de la familia tipográfica:

    a) usar una de las fuentes más comunes que muy probablemente tenga instalada el usuario (Arial, Verdana, Trebuchet, Georgia, san serif, Times)


    b) subir nuestra propia fuente con una propiedad nueva que se llama @font-face.


    c) usar una fuente de Google Fon.


    2. Font-size: Define el tamaño de la fuente y el valor se puede escribir en pixels o en ems.


    3. Color: Define el color de la tipografía. Los colores se pueden escribir de 3 formas distinas:


    • con sistema hexadecimal (#FF0000 (es rojo))


    • Con los nombres de los colores (más limitado) (black, red, Green)


    usando RGB, esta paleta permite agregar el canal alfa para hacer transparencias


    4. Width: Define el ancho de un elemento,el valor se puede escribir en:


    • pixels


    • ems


    • porcentaje


    5. Max-width o min-width: Definen el ancho máximo o mínimo de un elemento


    6. Height: Define el alto de un elemento, el valor se puede escribir en:


    • pixels


    • ems


    • porcentaje.


    7. Max-height o min-height: Definen el alto máximo o mínimo de un elemento. Definen el alto máximo o mínimo de un elemento. Muy importante en sitios adaptables


    8.Padding:Es la distancia desde el borde de un elemento hasta su contenido.


    9.Margin:Es la distancia entre un elemento y otro.


    10.Border:Define el borde de un elemento, su color, su estilo y grosor.


    11.Background: Define los fondos de un objeto. El fondo puede ser una imagen o un color.
